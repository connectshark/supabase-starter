## 如何使用快速開發模板

### 直接使用模板
點擊右上角`Use this template`直接在自己的Github中使用模板，或是[點擊這裡](https://github.com/new?template_name=supabase-starter&template_owner=connectshark)使用模板

### 專案初始化

```js
npm install
```
### 快速開發

```js
npm run dev
```

### 生產版本

```js
npm run build
```
